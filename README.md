# testng-cucmber
An example automation framework using TestNg &amp; Cucumber

## Precondition
- Setup JDK8, Maven 3
- Google Chrome: current version 88
- IDE IntelliJ IDEA. Plugin: Lombok, Cucumber.

## Run test
##### Method 1: Maven
Run command ```mvn test```

##### Method 2: Use IDE for execution or debugging
Run App.java
